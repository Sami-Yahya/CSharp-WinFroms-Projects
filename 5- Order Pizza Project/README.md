# 🍕 Order Pizza - Windows Forms Project

## 📖 Overview

A simple pizza ordering application built with **C# Windows Forms** as part of the **_“Course 14 - C# - Level 1”_** with **Dr. Mohammed Abu-Hadhoud**  
👉 [www.programmingadvices.com](https://www.programmingadvices.com).

The application allows users to customize their pizza order by selecting:

* 📏 Pizza size
* 🍕 Crust type (thickness)
* 🧀 Additional toppings
* 🍽️ Dining option (Eat In / Take Away)
* 🔢 Quantity

After making selections, the application displays an order summary along with the total price.

---

## ✨ Features

* Select pizza size
* Choose crust type
* Add optional toppings
* Select dining location
* Adjust quantity
* View order summary
* Calculate total price
* Place order with a single click
* Reset the form and start a new order
