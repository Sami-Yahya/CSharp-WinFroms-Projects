# 🧮 Simple Calculator - Windows Forms Project

## 📖 Overview

A simple calculator application built with **C# Windows Forms** as my own practice project of the **_“Course 14 - C# - Level 1”_** with **Dr. Mohammed Abu-Hadhoud**  👉 [www.programmingadvices.com](https://www.programmingadvices.com).

The application performs the most common mathematical operations while providing input validation to prevent invalid calculations and user mistakes.

Unlike scientific calculators, this project evaluates one operation at a time rather than full mathematical expressions.

---

## ✨ Features

### ➕ Basic Mathematical Operations

The calculator supports:

* Addition
* Subtraction
* Multiplication
* Division
* Exponentiation (Power)

---

### 🛡️ Input Validation

The application handles many edge cases to ensure a smooth user experience and prevent invalid input.

Examples include:

* Preventing an operator from being entered before a number
* Preventing multiple decimal separators in the same number
* Preventing consecutive operators
* Restricting direct typing inside the display textbox
* Preventing incomplete calculations
* Handling invalid user actions gracefully

---

### 🎯 User-Friendly Interface

* Large calculator buttons
* Clear display area
* Simple and intuitive layout
* Fast calculation results
