# ⏱️ Timer & Stopwatch - Windows Forms Project

## 📖 Overview

A time management application built with **C# Windows Forms** as my own practice project of the **_“Course 14 - C# - Level 1”_** with **Dr. Mohammed Abu-Hadhoud**  
👉 [www.programmingadvices.com](https://www.programmingadvices.com).

The project combines two tools in a single application:

* ⏱️ Stopwatch
* ⏳ Countdown Timer

All timing functionality is implemented using the **Windows Forms Timer Control**.

---

## ✨ Features

### ⏱️ Stopwatch

A stopwatch that tracks time with high precision, including:

* Hours
* Minutes
* Seconds
* Fractions of a second

Additional features:

* 📝 Record lap times using a ListBox
* ▶️ Start the stopwatch
* ⏸️ Pause the stopwatch
* 🔄 Reset the stopwatch

---

### ⏳ Countdown Timer

A customizable countdown timer that allows users to:

* Set the desired countdown duration
* Start and stop the timer
* View the exact time at which the countdown will finish

When the countdown reaches zero:

* 🔔 An alarm sound is played
* 📢 A notification informs the user that the timer has ended
* 🔁 The user is asked whether to restart the timer or stop it

Additional features:

* 🔇 Mute/stop timer ticking sound button

---

### ⚡ Preset Timers

A collection of predefined timer presets that allow users to quickly select commonly used countdown durations.

Benefits:

* Faster timer setup
* Convenient one-click selection
* Improved user experience
