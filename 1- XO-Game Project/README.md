# ❌⭕ Tic-Tac-Toe Game - Windows Forms Project

## 📖 Overview

A Tic-Tac-Toe (XO) game built with **C# Windows Forms** as part of the **_“Course 14 - C# - Level 1”_** with **Dr. Mohammed Abu-Hadhoud**  
👉 [www.programmingadvices.com](https://www.programmingadvices.com).

The application supports both **Player vs Player** and **Player vs Computer** modes, with multiple difficulty levels for the computer opponent.

---

## ✨ Features

### 👥 Player vs Player

* Two players can play on the same computer.
* Players choose their moves by clicking the desired cell.
* The game automatically detects:

  * 🏆 Winning conditions
  * 🤝 Draw situations
* Once the game ends, all game buttons are disabled to prevent further moves.

### 🤖 Player vs Computer

* Play against the computer.
* The computer takes its turn when the **Computer Turn** button is pressed.
* Two difficulty levels are available:

#### 🟢 Easy Mode

* The computer selects a random available position.
* Easy to beat and suitable for beginners.

#### 🔴 Hard Mode

The computer follows a simple strategy:

1. 🏆 Check if it has a winning move and play it.
2. 🛡️ If not, block the player's winning move.
3. 🎲 If neither exists, choose a random available position.

This makes the game significantly more challenging than Easy Mode.
