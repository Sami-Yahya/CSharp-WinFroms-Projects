# 🎂 Age Calculator - Windows Forms Project

## 📖 Overview

An Age Calculator application built with **C# Windows Forms** as my own practice project of the **_“Course 14 - C# - Level 1”_** with **Dr. Mohammed Abu-Hadhoud**  👉 [www.programmingadvices.com](https://www.programmingadvices.com).

The application calculates a user's age based on their date of birth and displays the result in multiple time units, providing a detailed breakdown of the user's lifetime.

The date of birth is selected using a **DateTimePicker** control for improved usability and input accuracy.

---

## ✨ Features

### 📅 Date of Birth Selection

* Select a birth date using a DateTimePicker control.
* Prevents users from selecting a future date.
* Ensures valid age calculations.

---

### 🧮 Age Statistics

The application calculates the user's age in:

* 🎉 Years
* 📆 Months
* 📅 Weeks
* ☀️ Days
* ⏰ Hours
* ⌛ Minutes
* ⚡ Seconds

---

### 🎯 Exact Age Calculation

In addition to the total statistics, the application also displays the user's exact current age in:

* Years
* Months
* Days

Example:

```text
22 Years, 4 Months, 15 Days
```

This provides a more accurate representation of age than simply showing the total number of years.
